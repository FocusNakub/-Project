# บ่วง ลวง บาป | Production Portal - Classic Window

ไฟล์นี้เป็นเวอร์ชันหน้าต่าง/หน้าตาแบบเดิม พร้อมใช้อัปขึ้น GitHub Pages

## วิธีลง GitHub Pages
1. แตก ZIP
2. อัปโหลด `index.html` ไปไว้หน้าแรกของ Repository
3. เปิด Settings > Pages
4. เลือก Branch: main และ Folder: /root
5. รอ 1-3 นาที แล้วเปิดลิงก์เว็บ
